Monster Survivors

Monster Survivors is an intense survival action game where you control a hero fighting against endless waves of monsters. Level up, choose powerful abilities, and survive as long as you can in this challenging roguelike experience.

Game Controls

  • WASD / Arrow Keys - Move character
  • Mouse - Aim abilities
  • Space - Dash/Dodge
  • 1-4 - Select abilities
  • ESC - Pause game

Gameplay Tips

  • Prioritize movement speed and area damage abilities early game
  • Keep moving to avoid being surrounded by monsters
  • Pick up experience orbs to level up and get stronger
  • Balance between offensive and defensive abilities
  • Watch your health bar and try to maintain distance from large groups

Mahjong

Classic Mahjong solitaire game with beautiful tiles and layouts. Match pairs of identical tiles to clear the board in this relaxing puzzle experience.

Game Controls

  • Mouse Click - Select tiles
  • ESC - Pause game
  • R - Restart level
  • H - Show hint

Gameplay Tips

  • Start with tiles that are completely exposed
  • Look for matching pairs that will unblock the most tiles
  • Use hints sparingly to avoid getting stuck
  • Plan ahead and think about which matches will open up more possibilities

Space Shooter

Classic space shooting game where you pilot a spaceship through dangerous asteroid fields and enemy waves. Collect power-ups and survive as long as possible in this action-packed space adventure.

Game Controls

  • WASD / Arrow Keys - Move spaceship
  • Space - Shoot
  • P - Pause game
  • ESC - Menu

Gameplay Tips

  • Keep moving to avoid enemy fire
  • Collect power-ups to enhance your weapons
  • Watch out for asteroid patterns
  • Use the edges of the screen strategically
  • Focus on survival over scoring initially

Color Match

A challenging puzzle game where you need to match colors in the correct sequence. Test your memory and reaction time as you progress through increasingly difficult levels. Can you master all the color combinations?

Game Controls

  • Mouse Click - Select colors
  • ESC - Pause game
  • R - Restart level

Gameplay Tips

  • Pay attention to the color sequence
  • Take your time on harder levels
  • Try to memorize patterns
  • Practice makes perfect
  • Use visual mnemonics to remember sequences
